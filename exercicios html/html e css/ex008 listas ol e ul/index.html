Também é possivel escolher se você deseja que a ordem seja 123 ou 321, segue o exemplo:

  1. Acordar
  2. Ligar para o João
  3. Escovar os Dentes
  4. Tomar Café
  5. Lavar Roupa

Também é possivel escolher se você deseja que a ordem seja A ou I, segue o exemplo:

  1. Acordar
  2. Ligar para o João
  3. Escovar os Dentes
  1. Acordar
  2. Ligar para o João
  3. Escovar os Dentes
