Usando links externos

A tag a (ancora) serve para transforma uma palavra ou frase em link para levar a um outro site ou pagina, além disso podemos complementar a tag a com target _blank para abrir em uma nova guia no navegador e rel= external para dizer que o site que vai ser aberto é de outro site externo, diferente do site que o visitante está, por exemplo:

Você pode acessar a minha pagina no Facebook

Você pode acessar a minha pagina no Instagram

Você pode acessar a minha pagina no Youtube
